Project Fuga

Project Fuga is een project waarmee je via het web kassa’s, bonprinters, etc. kan aansturen. Ze zijn al de hele wereld overgevlogen om dit te demo-en. Demo gaat erg soepel.

Bonnen printer

D.m.v. een API kan je dit sturen naar de bonnenprinter.

De taal die deze printers gebruiken is nog ouder dan HTML (jaren 70). Deze taal is gebaseerd op ASCII. Er zit dus ook geen ë of euroteken in.

Door HTML te vertalen naar deze taal kan je dit naar de printer sturen. Unicode kan dit probleem oplossen.

Hoe voeg je dan afbeeldingen toe?

Maak er eerst een bitmap van → hoe meer, hoe donkerder de pixel.

Kassaladen

Alle bonnenprinters hebben een speciale poort. Deze geeft een puls die de kassalade opent.

Library

Dit kan dus allemaal met een JavaScript-library. Het is alleen niet zo makkelijk als importeren.

Elke printer werkt namelijk anders. Ze ondersteunen niet allemaal dezelfde ASCII-codes. Sommige hebben er veel, andere maar vijf. Het aantal pixels per regel verschilt ook.

Hiervoor zijn libraries die dit vertalen naar het formaat dat een specifieke printer begrijpt.

Bonnenprinters zijn een rotzooi.

Barcode(scanner)

Barcodes zijn gewoon een reeks cijfers en een scanner is eigenlijk een toetsenbord: het “typt” de cijfers razendsnel achter elkaar.

Ook dit is een rommeltje, want elk land gebruikt natuurlijk weer zijn eigen code.

Over 2 jaar zijn er geen verpakkingen meer met een barcode.
