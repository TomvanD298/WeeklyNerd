Demo 2 - In-page transitions

html {
  scroll-behavior: smooth;
}

Zorgt ervoor dat als de pagina auto scrolled, kan de browser het automatisch scrollen. 

@media (prefers-reduced-motion: no-preference) {
  /* animaties zijn ok */
}

Niet iedereen vindt dit fijn, dus dit is code om het te checken of iemand dit aangegeven heeft. 

#myItem {
  scroll-margin-top: 100px;
}

Zorgt ervoor dat er wat meer ruimte is om te scrollen zonder dat er dingen over elkaar heen gaan liggen. 

#my-target {
  outline: 10px solid red;
  transition: 1s ease-in-out outline;
}

Legt een focus op het element.
